Hello, my name is Alexis Clerc--Beaufort and I am an IT student at the UTI at Bordeaux-Gradignan


I'm going to present you my feedback on the project "hyperstellar"

This project involved creating a board game from scratch called Hyperstellar. The objective
was to reach 150 victory points before the opponent. Players placed tokens in different
rooms on the board each turn. The game also featured a resource management system for
energy and water, as well as a card system offering various bonuses. The main goal was to
design the code architecture based on our needs and then create and code an interface for
gameplay. The project took one and a half months to complete and encompassed all the
object-oriented development concepts from the second semester.
